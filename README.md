# Nomenclatura TMPV

## Backend (FastAPI)

```bash
cd backend
python -m venv venv
venv/Scripts/activate  # Windows
pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend (Vite + Vue)

```bash
cd frontend
npm install
npm run dev
```
